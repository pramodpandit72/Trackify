# Trackify

Trackify is a comprehensive MERN stack application designed to streamline fitness tracking, trainer management, job applications, and user engagement. It offers a seamless experience for both trainers and users, integrating advanced features for communication, booking, and progress monitoring.

> https://trackify-two-beta.vercel.app/

## Features

- User authentication and role-based access (Admin, Trainer, User)
- Trainer listing, reviews, and booking management
- Real-time chat between users and trainers
- Job board for trainers and application management
- Exercise library and progress tracking
- Contact and support system
- Responsive, modern UI with multi-step forms and dashboards

## Tech Stack

- **Frontend:** React, Vite, Context API, Tailwind CSS
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Other:** JWT Authentication, Multer (file uploads), Email integration

## Folder Structure

- backend – REST API, controllers, models, routes, middleware, and utilities
- frontend – React app, components, pages, context, and assets
